¿Qué son los Lenguajes de Marcas?

Son un lenguaje que permite codificar un documento, combinando la información del texto con marcas o anotaciones relativas a a la estructura o a la forma de representar el propio texto.

Clasificación de los Lenguajes de Marcas
Tipo Descripción Ejemplo
Lenguajes Orientados a Presentación Codifican como ha de presentarse el documento y son usados tradicionalmente por los procesadores de texto. Indican el formato del texto o tipografía, sin especificar su estructura, por ejemplo: aumentar el tamaño de la fuente,centrar o cambiar a negrita. El software que representa el documento debe interpretar el código en el mismo orden en que aparece. RTF (RichText Format), Docbook (derivados de SGML)
Lenguajes Descriptivos Se usan para describir la estructura lógica del documento y la descripción del contenido. Las marcas describen qué es lo que se está representando. Las marcas no determinan el procesamiento del documento de manera fija, ya que dicho procesamiento se determina a partir de las necesidades concretas, y se beneficia de la estructura lógica del documento caracterizada a través de sus marcas. SGML, XML
Lenguajes Híbridos Son lenguajes que contienen marcas de los dos tipos anteriores indistintamente. HTML, XHTML

HTML (HyperText Markup Language, 1989)

Es un lenguaje orientado expresamente para la presentación o visualización de información en la web

Ventajas Desventajas
Es muy simple y sencillo de usar Presenta información estática
No requiere herramientas especiales para su edición Esta orientado funcionalmente a la representación de los datos y no a su estructura
Esta muy difundido en la web No es adecuando pata intercambio de datos (interactividad)
No es extensible: no se puenden definir nuevas etiquetas

Código de ejemplo


<HTML>
<HEAD>
<TITLE>Mi primer documento HTML</TITLE>
</HEAD>
<BODY>
<P>¡Hola mundo!</P>
</BODY>
</HTML>

XML (Extensible Markup Language, 1996)

Está diseñado para almacenar y transportar datos entre aplicaciones. Por eso, simplifica la compartición de datos entre programas y la realización de cambios en las configuraciones de los sistemas. Permite también crear otros lenguajes a partir de él, como el XHTML, por lo que se dice que es un metalenguaje.
No existen etiquetas predefinidas, sino que es el usuario quien las crea. Se centra en qué son los datos (es un lenguaje descriptivo), mientras que el HTML se encarga de cómo se tienen que visualizar esos datos en un navegador.


Código de ejemplo


<Edit_Mensaje>
<Mensaje>
<Remitente>
<Nombre>Nombre del remitente</Nombre>
<Mail> Correo del remitente </Mail>
</Remitente>
<Destinatario>
<Nombre>Nombre del destinatario</Nombre>
<Mail>Correo del destinatario</Mail>
</Destinatario>
<Texto>
<Asunto>
ASUNTO
</Asunto>
<Parrafo>
Este es mi documento con una estructura muy sencilla ...
</Parrafo>
</Texto>
</Mensaje>
</Edit_Mensaje>
